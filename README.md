# Система компьютерной алгебры

## Описание файлов проекта

### Основные файлы приложения:
- `gui_app.py` - главный файл GUI приложения (Tkinter)
- `natural.py` - модуль для работы с натуральными числами
- `integer.py` - модуль для работы с целыми числами
- `rational.py` - модуль для работы с рациональными числами (дробями)
- `polynomial.py` - модуль для работы с многочленами

### Вспомогательные файлы:
- `test_functions.py` - тесты функциональности
- `build_exe.py` - скрипт автоматической сборки EXE
- `README.md` - эта инструкция

---

## Возможные проблемы и решения

### Ошибка при запуске
- Убедитесь, что Windows обновлен
- Попробуйте запустить от имени администратора

### Антивирус блокирует файл
- Некоторые антивирусы могут блокировать EXE файлы, созданные PyInstaller
- Добавьте файл в исключения антивируса

## Возможности приложения

### Натуральные числа
- Сложение, вычитание, умножение, деление
- НОД (наибольший общий делитель)
- НОК (наименьшее общее кратное)

### Целые числа
- Все операции с учетом знака
- Деление с остатком
- Определение знака числа

### Рациональные числа (дроби)
- Сложение, вычитание, умножение, деление
- Автоматическое сокращение дробей
- Формат ввода: `3/4` или просто `7`

### Многочлены
- Операции над многочленами с рациональными коэффициентами
- Вычисление производной
- НОД многочленов
- Формат ввода: `1,2,3` означает 3x² + 2x + 1

---

## Технические детали

### Использованные технологии:
- **Python** - язык программирования
- **Tkinter** - библиотека для GUI (встроена в Python)
- **PyInstaller** - инструмент для создания EXE файлов

### Особенности реализации:
- Длинная арифметика реализована "с нуля"
- Числа хранятся как списки цифр
- Поддержка произвольно больших чисел

## Запуск
Скачайте архив и просто запустите файл gui_app.py

---

## Сборка
1. Убедитесь, что Python установлен на вашем компьютере, а так же все библиотеки
2. Запустите файл `build_exe.py`:
   ```
   python build_exe.py
   ```
3. Дождитесь завершения сборки
4. Готовый исполняемый файл будет в папке `dist/ComputerAlgebra.exe`
